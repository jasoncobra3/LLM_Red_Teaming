Quick Tips
  • Keys are saved to the database and applied to the current session immediately — no server restart needed.
  • Use Test Connection to verify your key works before running attacks.
  • You only need one provider configured to start red-teaming. The same provider can be used as both attacker and target.
  • For Azure OpenAI, you need all three fields: API key, endpoint URL, and deployment name.
  • AWS Bedrock uses IAM credentials. Make sure the IAM user has bedrock:InvokeModel permissions.
